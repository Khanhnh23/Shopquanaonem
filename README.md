#  Shopquanaonem

Website bán quần áo nữ sử dụng React + ASP.NET Core Web API

---

##  Công nghệ sử dụng

- ReactJS
- ASP.NET Core Web API
- Bootstrap 5

---

##  Tính năng

- Xem sản phẩm
- Thêm giỏ hàng
- Đăng nhập / đăng ký 
- Quản lý sản phẩm (Admin)

---

## ⚙️ Chạy dự án

### Frontend

```bash
npm install
npm start
```
### Backend
cd shop-be
dotnet restore
dotnet run
